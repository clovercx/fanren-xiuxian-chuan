## 《凡人修仙传》—— 互动小说游戏

**一部从凡人到化神的完整修仙史诗，4小时爆肝开发，538个场景。**

改编自同名修仙小说，你扮演青云镇的一个打柴少年，在一次奇遇中踏入仙途。从凡人之躯一步步修炼到化神巅峰，在命运的洪流中逆天改命。

### 核心特色

📖 **十五章完整主线** — 凡人 → 练气 → 筑基 → 结丹 → 元婴 → 化神，538个精心编排的剧情场景，完整的修仙体验。

🔀 **丰富分支选择** — 每个关键节点都有多分支选项，你的每一个决定都影响属性和剧情走向。总计超过 200 个选项节点。

🏆 **三重结局** — 飞升天宫 / 守护此界 / 我命由我不由天，由你的最终选择决定命运。

⚔️ **特色Boss战** — 冰魄蟾·赤鳞蛟·血煞宗主·四九小天劫·朱雀·青龙·白虎·圣教教主，每个Boss都有独特战斗机制。

🪄 **核心宝物贯穿全剧** — 天火之种·玄天镜·玄元鼎，三件至宝陪伴你从筑基走向化神。

📝 **留言墙 + 游玩统计** — 留下你的修仙感悟，看看多少道友与你同行。

🎯 **在线即玩** — 浏览器打开即可畅玩，无需下载安装。
